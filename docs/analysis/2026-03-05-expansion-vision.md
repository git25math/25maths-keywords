# 扩展愿景: 从词汇工具到双语数学学习支撑平台

> **创建日期**: 2026-03-05
> **最后更新**: 2026-03-07（v3.0 — 基于 v1.12.5 进展全面更新）
> **状态**: 活跃文档

## 背景

25Maths-Keywords 从一个简单的词汇闪卡工具起步，经过 v1.0 到 v1.9.8 的 20 个大版本迭代，已发展为覆盖 3 个考试局、264 级、2,200+ 词、4,100+ 真题的双语数学学习平台。本文记录平台的四层扩展愿景及当前落地进度。

---

## 四层扩展模型

```
Layer 4: Assessment  诊断测试 / 模拟考试 / 成绩预测        [已完成 70%]
Layer 3: Practice    练习题 / 真题 / 错题本 / 智能推荐      [已完成 95%]
Layer 2: Concept     知识卡片 / 公式卡 / 考纲驱动           [已完成 70%]
Layer 1: Vocabulary  词汇闪卡 / SRS / 七模式 / 教师系统     [已完善]
```

---

### Layer 1: Vocabulary — 已完善

v1.3.0 奠基，经多轮打磨已高度成熟。

**已实现能力**:
- 7 种学习模式: Study / Quiz / Spell / Match / Battle / Review / Daily Challenge
- SRS 8 级艾宾浩斯复习系统 + 星级计分（learningPct + masteryPct 双指标）
- 学习路径引导（主线三步 + 辅助三模式 + 进度标记 + 错词即时复习）
- 3 考试局/课程: CIE 0580（50 级 390 词）+ Edexcel 4MA1（41 级 308 词）+ 25Maths Y7-11（173 级 1,502 词）
- 教师管理系统: 班级/学生 CRUD + 批量建号 + 作业布置 + 成绩分析 + 站内通知
- 云端排行榜（多维度 Scope）+ 段位系统 + 学习连续天数
- 首屏优化 ~47-55KB gzip（按 board 懒加载 + esbuild 打包）
- 新手引导 Tour + 深色模式 + 音效系统

**评估**: 此层已无重大缺口，进入维护模式。

---

### Layer 2: Concept Cards — 已完成 60%

**愿景**: 每个考纲知识点配备公式卡、知识卡、图示、经典例题，形成结构化知识体系。

**已实现** (v1.6.0 - v1.9.5):
- CIE 考纲骨架: 9 章 72 知识点，517 词 + 1,488 题映射 (v1.6.0)
- Edexcel 考纲骨架: 6 章 39 知识点，385 词 + 936 题映射 (v1.7.0)
- 知识卡片内容种子: 111 张双语知识卡（CIE 72 + Edexcel 39），含知识要点 + 经典例题
- 知识卡片质量审核: 60 张 ZH 补全 + Edexcel 30 张翻译 + 32 张低质量卡修复，平均 ZH 比率 28.5% -> 54.8% (v1.9.3-v1.9.5)
- LaTeX 数学公式渲染: pqRender + KaTeX，CIE 14 处 + Edexcel 11 处 (v1.7.1)
- 知识点模块纠错: Vocabulary/Practice 双路径纠错按钮 (v1.6.1)
- 考纲内容超管内联编辑器 (v1.7.2 内置于 section_edits 表)

**未实现**:
- [ ] Formula Builder — 拖拽组件构建公式（交互模式）
- [ ] Concept Map — 知识点关联图（可视化）
- [ ] 经典例题 AI 批量生成（10C Phase B，~300 道 worked examples）
- [ ] 知识卡片富文本编辑增强（目前为 Markdown + KaTeX，缺图片嵌入）

**评估**: 知识点骨架和内容已落地，但缺乏交互式学习模式（Formula Builder / Concept Map）。这两项为"锦上添花"，非核心阻塞。

### Layer 3: Practice — 已完成 90%

**愿景**: 从选择题练习到真题训练，覆盖完整的"做题"闭环。

**已实现** (v1.4.0 - v1.9.8):

| 能力 | 版本 | 规模 |
|------|------|------|
| 选择题练习 | v1.4.0 | 2,424 题（CIE 1,488 + Edexcel 936） |
| 真题引擎试点 | v1.8.0 | 代数章 793 题 LaTeX->KaTeX |
| 全量真题扩展 | v1.9.0-v1.9.2 | 4,107 题 228 套卷，覆盖 66/72 sections（92%） |
| 题型归纳 | v1.9.6 | 12 种 Command Words 分类 + 筛选 |
| 词汇联动 | v1.9.7 | 真题关联词汇 + 跳转词汇卡组 |
| 智能推荐 | v1.9.8 | getSectionHealth 统一评分 + 首页推荐学习区 + 知识点健康度环 |
| 套卷系统 | v1.9.0-v1.9.1 | 套卷浏览/考试模式 + 倒计时 + 成绩持久化 |
| 错题本 | v1.8.0 | 自动收集 + 定期复习提醒 |
| 题目纠错 | v1.5.0 | 用户报错 + 超管编辑器 |
| TikZ 图表渲染 | v1.8.4 | SVG 内联 + 暗色模式反色 + PNG 兜底 |

**未实现**:
- [ ] 分步解题（step-by-step solution）— 需 AI 辅助或人工编写
- [ ] 难度自适应（根据正确率动态调整题目难度）
- [ ] CIE 3 个 section 缺练习题（1.6 Order of operations / 1.7 Bounds / 4.8 Constructions）
- [x] 教师端布置练习题作业 → v1.12.4
- [ ] Edexcel 真题引擎（目前仅 CIE 有真题数据）

**评估**: 此层是过去 8 个版本的核心投入区，从"零基础"推进到 4,100+ 题 + 套卷 + 智能推荐，进展显著。剩余项中"分步解题"和"难度自适应"是最有价值的方向。

### Layer 4: Assessment — 已完成 70%

**愿景**: 以诊断测试和模拟考试闭合学习反馈环。

**已实现** (v1.10.1 - v1.12.5):
- [x] 诊断测试: 跨知识点 20 题加权选题 + 知识点逐项得分 + 弱项推荐 + 历史记录 → v1.10.1
- [x] 模拟卷生成器: 目标分数/侧重模式自动组卷 + 轮询选题 + 难度排序 + 知识点分析 → v1.10.3
- [x] PWA 离线: Service Worker 三层缓存 + manifest + 安装提示 + 离线检测 → v1.11.0
- [x] 4 维掌握度系统: Vocab/Practice/Papers/Retention 追踪 + 复习计划推荐 → v1.12.0
- [x] 教师练习题作业: MCQ 选题 + KaTeX 答题 + 结果存储 → v1.12.4

**未实现**:
- [ ] 成绩预测: 基于练习数据预测考试等级
- [ ] 服务端评分: Edge Function 防作弊
- [ ] 多题型混合: 选择 / 填空 / 简答
- [ ] 评分 rubric: Mark scheme 对照评分

**评估**: 诊断测试和模拟卷已落地，PWA 离线已实现。剩余项中"成绩预测"最有差异化价值。

3. **教师生态**: 作业系统 + 班级管理 + 成绩分析形成学校采购动力
